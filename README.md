This GitHub Repository relates to the following paper: **Agnostic Inter-cell Interference Mitigation for Heterogeneous 5G O-RAN Networks Using Reinforcement Learning (AIIM)**

**System description:**
- 1 Hardware SDR-based (B210) gNodeB: gNB1
    - 1 SDR-based (B210) eMBB user: UE0
- 2 Virtual ZMQ-based gNodeBs: gNB2, gNB3
    - 1 virtual eMBB user and 1 virtual mMTC user per gNB
 
**For viewing the full system interference demo from the GitHub GUI:**
- Download the raw file to your system [~229MB] and use a standard video player that supports MP4
